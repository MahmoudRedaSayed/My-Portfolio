# Mahmoud Reda Sayed — Portfolio

Personal portfolio website for Mahmoud Reda Sayed, Senior Backend Developer at PaySky.

**Live:** [my-portfolio-nine-mu.vercel.app](https://my-portfolio-nine-mu.vercel.app/)

## Stack

- React 17
- React Bootstrap
- React Router
- Typewriter Effect
- React TSParticles
- GitHub Calendar

## Sections

- Home — introduction and social links
- About — experience, education, tech stack, and GitHub activity
- Projects — featured and additional project showcase

## Run locally

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```
